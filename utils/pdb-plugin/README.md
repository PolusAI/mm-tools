# Download PDB file (0.1.0)

Download PDB file

## Options

This plugin takes 1 input arguments and 1 output argument:

| Name                   | Description                                  | I/O    | Type   |
|------------------------|----------------------------------------------|--------|--------|
| config                 | PDB config                                   | Input  | string
| output_pdb_path        | output pdb path                              | Input  | string
